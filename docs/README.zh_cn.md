# 更改农场洞穴 - 星露谷物语

## 介绍与使用

脚本会检查你的农场洞穴类型，并询问你是否要更改为另一种类型(蝙蝠或者蘑菇)；
请将脚本文件“script.ps1”和“START.bat”复制到要操作的存档文件夹内，然后运行“START.bat”即可，存档处理后会一直生效，并生成一个“.bak”为后缀的备份，之后可以删除该模组也可以多次修改；

注意，模组仅能在Windows平台操作(或者支持PowerShell 5.1的操作系统)，如果你的存档位于Android手机或其他操作系统，请将存档文件发送到一个Windows设备上操作，处理完成后再转移回原设备即可；
